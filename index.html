C Programming Language Documentation

Introduction

Welcome to the C Programming Language documentation. This guide will help you understand the basics of C, one of the most powerful and widely used programming languages.

Overview

C is a powerful general-purpose programming language that has been widely used in various applications since its creation. Developed in the early 1970s by Dennis Ritchie at Bell Labs, C has become one of the most enduring and influential programming languages. It has shaped many modern languages, including C++, C#, and Java, and continues to be a cornerstone in the world of software development.

Setup

To start programming in C, you'll need to set up your development environment. This involves installing a C compiler and optionally choosing an Integrated Development Environment (IDE) to write, compile, and debug your code. Here's a step-by-step guide to get you started:

  • Windows
    • MinGW: Download and add to PATH.
    • Visual Studio: Download and select "Desktop development with C++".
  • macOS
    • Xcode: Download from the Mac App Store.
    • Command Line Tools: Install via Terminal with xcode-select --install.
  • Linux
    • GCC: Install via Terminal with sudo apt update && sudo apt install build-essential (Ubuntu).

Choosing an IDE

The IDE I personally recommend is Visual Studio Code: After installing Visual Studio Code Install the "C/C++" extension.

Setting Up Your First C Program

File Extension and Opening in VS Code

To start coding in C, create a file with a .c extension. This tells your compiler that the file contains C source code.

Creating a File:

  • Create a new file and save it as hello.c.

Opening the File in VS Code:

  • In Visual Studio Code, go to File > Open File... and select your hello.c file.
  • You can also open the integrated terminal in VS Code by pressing Ctrl+`.

Understanding the #include Directive

The #include directive in C is used to include the contents of a file or library in your program. This is often used to include standard libraries that provide useful functions.

#include <stdio.h>

The #include <stdio.h> directive tells the compiler to include the Standard Input Output library, which contains functions like printf.

The main Function

Every C program must have a main function. This is the entry point where the execution of the program begins.

int main() {
        // Your code goes here
        return 0;
        }

The main function returns an integer value. By convention, returning 0 indicates that the program executed successfully.

Using the printf Function

The printf function is used to print text to the console. It's a part of the Standard Input Output library that we included with #include <stdio.h>.

#include <stdio.h>

        int main() {
        printf("Hello, World!\n");
        return 0;
        }

Writing Your Program:

  • In your hello.c file, type the code above.

Compiling and Running Your Program:

  • Open Terminal or Command Prompt, navigate to your file's directory, and run:
    • gcc hello.c -o hello
        ./hello
  • You should see the output: Hello, World!

Formatted Input/Output

Formatted Output with printf

The printf function is used to output formatted text to the console. It allows you to format strings and display variables of various types.

Basic Syntax:

printf("format string", argument1, argument2, ...);

Format Specifiers: The format string can contain placeholders (format specifiers) that are replaced by the values of the arguments.

  • %d for integers
  • %f for floating-point numbers
  • %c for characters
  • %s for strings

Example:

int age = 25;
    float gpa = 3.75;
    char grade = 'A';
    char name[] = "Alice";

    printf("Name: %s\n", name);
    printf("Age: %d\n", age);
    printf("GPA: %.2f\n", gpa);
    printf("Grade: %c\n", grade);

Formatted Input with scanf

The scanf function is used to read formatted input from the console. It reads input according to the format specifiers and stores the values in the provided variables.

Basic Syntax:

scanf("format string", &variable1, &variable2, ...);

Ampersand (&): The address-of operator (&) is used to pass the address of the variables where the input will be stored.

Example:

int age;
    float gpa;
    char grade;
    char name[50];

    printf("Enter your name: ");
    scanf("%s", name);

    printf("Enter your age: ");
    scanf("%d", &age);

    printf("Enter your GPA: ");
    scanf("%f", &gpa);

    printf("Enter your grade: ");
    scanf(" %c", &grade);  // Note the space before %c to consume any leftover newline

    printf("Name: %s, Age: %d, GPA: %.2f, Grade: %c\n", name, age, gpa, grade);

Expressions

Expressions in C are combinations of variables, constants, and operators that compute a value.

Arithmetic Expressions

Arithmetic expressions perform mathematical operations using operators like +, -, *, /, and %.

Example:

int a = 5, b = 3;
    int sum = a + b;      // sum is 8
    int difference = a - b;  // difference is 2
    int product = a * b;  // product is 15
    int quotient = a / b; // quotient is 1
    int remainder = a % b; // remainder is 2

Relational Expressions

Relational expressions compare values using operators like ==, !=, <, >, <=, and >=.

Example:

int x = 10, y = 5;
    int result;

    result = (x == y); // result is 0 (false)
    result = (x != y); // result is 1 (true)
    result = (x > y);  // result is 1 (true)
    result = (x < y);  // result is 0 (false)
    result = (x >= y); // result is 1 (true)
    result = (x <= y); // result is 0 (false)

Logical Expressions

Logical expressions combine multiple conditions using logical operators like && (AND), || (OR), and ! (NOT).

Example:

int a = 1, b = 0;
    int result;

    result = (a && b); // result is 0 (false)
    result = (a || b); // result is 1 (true)
    result = (!a);     // result is 0 (false)
    result = (!b);     // result is 1 (true)

One-Dimensional Arrays

An array is a collection of elements of the same type stored in contiguous memory locations.

Declaring and Initializing Arrays

You can declare an array by specifying its type and size. You can also initialize it at the time of declaration.

Declaration:

int numbers[5]; // declares an array of 5 integers

Initialization:

int numbers[5] = {1, 2, 3, 4, 5}; // initializes the array with 5 values

Accessing Array Elements

Array elements are accessed using their index, which starts from 0.

Example:

int numbers[5] = {1, 2, 3, 4, 5};

    printf("First element: %d\n", numbers[0]); // prints 1
    printf("Second element: %d\n", numbers[1]); // prints 2

    // Modifying an array element
    numbers[2] = 10;
    printf("Modified third element: %d\n", numbers[2]); // prints 10

Iterating Through Arrays

You can use loops to iterate through arrays and perform operations on each element.

Example:

int numbers[5] = {1, 2, 3, 4, 5};
    int sum = 0;

    for (int i = 0; i < 5; i++) {
        sum += numbers[i];
    }

    printf("Sum of elements: %d\n", sum); // prints the sum of all elements
